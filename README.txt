Summary:
mtcars Explorer provides the means to do basic exploration of the distributions of each variable in the mtcars dataset in the datasets package.

User Description:
1: start the app.
2: Choose the variable to explore in the drill-down list "Choose a variable".
3: The distribution of the selected variable is shown to the right.
4: The red vertical line in the distribution plot shows mu, the current guess at the mean. 
4: Use the slider "Guess at the mean" to move mu between the min and the max values of the slected variable.
